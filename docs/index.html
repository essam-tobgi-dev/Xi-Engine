main.cpp 
class MyGame : public Xi::Application {
public:
    void OnInit() override {
        // Create player entity
        Entity player = GetWorld().CreateEntity("Player");
        auto& transform = GetWorld().AddComponent<Transform>(player);
        auto& mesh = GetWorld().AddComponent<MeshRenderer>(player);

        mesh.mesh = ResourceManager::GetMesh("player.obj");
        mesh.material = ResourceManager::GetMaterial("pbr");
    }
};

int main() {
    MyGame game;
    game.Run();
    return 0;
}

Powerful Features

Everything you need to build modern games

Entity Component System

Data-oriented ECS architecture with cache-friendly component storage, fast queries, and efficient iteration patterns for maximum performance.

  • Packed component arrays
  • O(1) entity lookups
  • Variadic template queries
  • Entity hierarchies

Lua Scripting

Full Lua scripting support for gameplay logic with native C++ bindings. Write game behavior without recompiling the engine.

  • Lua syntax support
  • Entity & component access
  • Input handling
  • Syntax highlighting editor

OpenGL Renderer

Modern OpenGL 4.5 rendering pipeline with PBR materials, dynamic lighting, and shader-based rendering.

  • PBR material workflow
  • Multiple light types
  • Texture mapping support
  • Render queue sorting

Physics System

Complete physics simulation with collision detection, rigid body dynamics, and collision callbacks for interactive gameplay.

  • AABB & sphere collision
  • Rigid body dynamics
  • Raycasting support
  • Collision callbacks

3D Audio

Spatial audio system with 3D positioning, distance attenuation, and listener orientation using MiniAudio backend.

  • 3D positional audio
  • Distance attenuation
  • Volume & pitch control
  • Looping support

Integrated Editor

Full-featured ImGui-based editor with scene hierarchy, component inspector, viewport, and script editor with syntax highlighting.

  • Scene hierarchy
  • Component inspector
  • Real-time viewport
  • Script editor
